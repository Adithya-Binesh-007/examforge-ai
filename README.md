# ExamForge AI

🚀 AI-powered question paper generator for students and educators.

ExamForge AI helps users generate professional question papers instantly using:

* Syllabus input
* Previous question paper patterns
* Custom prompts
* AI-powered formatting and difficulty balancing

---

## ✨ Features

* 🤖 AI Question Paper Generation
* 📚 Generate from Syllabus
* 📝 Generate from Previous Question Papers
* 🎯 Difficulty Selection
* 📄 PDF Export
* ☁️ Cloud Saved Papers
* 🔐 Secure Login System
* 📱 Responsive Modern UI
* 🌙 Dark Mode Support

---

## 🛠️ Tech Stack

* React
* TypeScript
* Tailwind CSS
* Supabase
* Vercel
* AI API Integration

---

## ⚡ How It Works

1. Login or create an account
2. Enter syllabus, prompt, or previous paper
3. Select difficulty and exam type
4. Generate AI-powered question paper
5. Save or download as PDF

---

## 🎯 Goal

ExamForge AI aims to simplify exam preparation and question paper creation for students, teachers, and institutions using modern AI technology.
